# 雁宝AI相机 - 完整设计规范提取

## 1. 色彩系统

### 主色调
- **紫色**: #A78BFA (RGB: 167, 139, 250)
- **粉红色**: #EC4899 (RGB: 236, 72, 153)  
- **浅粉色**: #F9A8D4 (RGB: 249, 168, 212)
- **深紫色**: #6B21A8 (RGB: 107, 33, 168)

### 渐变背景
- **垂直渐变**: #A78BFA → #EC4899 → #F9A8D4
- **应用**: 所有屏幕背景

### 霓虹灯效果
- **边框颜色**: #EC4899 (粉红)
- **发光效果**: 2-4px外阴影，模糊半径8-12px

## 9. 特殊效果

### 毛玻璃效果实现
```
背景色: Color.White.copy(alpha = 0.15f)
模糊: Modifier.blur(10.dp)
圆角: RoundedCornerShape(16.dp)
```

### 霓虹灯效果实现
```
边框: 2px 粉红色 (#EC4899)
外阴影: 0 0 12dp 粉红色，透明度50%
```

### 渐变背景实现
```
Brush.verticalGradient(
  colors = listOf(
    Color(0xFFA78BFA),  // 紫色
    Color(0xFFEC4899),  // 粉红色
    Color(0xFFF9A8D4)   // 浅粉色
  )
)
```
