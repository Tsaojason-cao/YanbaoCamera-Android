# SheYan UI 设计规范关键信息

## 核心品牌色
- 曜石黑: #0A0A0A (App 主背景色)
- 品牌粉: #EC4899 (核心强调色，高亮/选中状态/图标/按钮)
- 胡萝卜橙: #F97316 (奖励激励色，进度条等)
- 高斯模糊背景: rgba(10,10,10,0.6)，模糊半径 40dp

## 布局规范
- 72:28 黄金比例：上方 72% 内容展示区，下方 28% 控制面板区
- 安全边距：所有内容距离屏幕边缘至少 16dp

## 字体规范
- 中文字体：思源黑体 (Source Han Sans CN)
- 英文字体/数字：Montserrat
- 大标题 Display: 32sp
- 页面标题 Headline: 24sp
- 按钮/标签 Button/Label: 16sp
- 正文/说明 Body: 14sp

## IP 视觉元素
- 雁宝 (Yanbao)：兔+熊混合体，身穿 JK 制服，戴粉色蝴蝶结
- 熊掌印：快门按钮、导航栏中心按钮
- 兔耳：容器或装饰，模式拨盘选中标记、头像框、徽章等

## 组件规范
- 熊掌快门：72dp，Canvas 绘制，带呼吸灯动画和震动反馈
- 兔耳拨盘：Canvas 绘制兔耳标记，带居中吸附和弹跳动画
- 胡萝卜滑块：胡萝卜形状 Thumb，带气泡数值显示

## M4 编辑模块规范
- 设计图：EDIT_01_filter.png, EDIT_02_parallax.png, EDIT_03_beauty.png
- 功能：对已拍摄的照片进行后期处理
- 组件：
  1. 滤镜工具：91个滤镜，支持强度调节
  2. 2.9D 视差调节：通过滑块调整景深和位移效果
  3. 美颜塑形：20+参数，精细调整

## 底部导航栏
- 5个入口：首页、编辑、拍照、推荐、我的
- 中心"拍照"按钮：超大粉色熊掌印，带霓虹光晕，高于导航栏
- 其余图标：雁宝兔耳风格设计
