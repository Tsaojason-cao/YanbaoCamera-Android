## 技术实现要点

### 1. 顶部控制栏
- 使用 `Box` + `Modifier.background(Color.White.copy(alpha = 0.3f))` 实现毛玻璃效果
- 使用 `Row` 布局排列控制按钮
- 使用 `Icon` 显示图标，`Text` 显示文字

### 2. 中央预览区
- 使用 `AndroidView` 嵌入 `PreviewView`（CameraX）
- 使用 `Canvas` 或 `Box` + `Image` 绘制四个角落的装饰元素
- 装饰元素使用 `Modifier.align()` 定位到四个角落

### 3. 底部操作栏
- 使用 `Box` + `Modifier.background(Brush.horizontalGradient(...))` 实现粉紫渐变
- 使用 `Row` 布局排列操作按钮
- 拍照按钮使用 `Canvas` 绘制3层同心圆环
- 使用 `rememberInfiniteTransition` 实现呼吸动画

### 4. 库洛米装饰
- 使用 `Text` 显示 emoji（🐰 + 💗 + 🎀 + ⭐）
- 或者使用 `Image` 加载库洛米图片资源
- 使用 `Modifier.offset()` 调整位置
- 使用 `Modifier.scale()` 调整大小
- 使用 `Modifier.alpha(0.15f)` 设置 15% 透明度（根据设计规范）

### 5. 相册缩略图
- 使用 `AsyncImage`（Coil）加载最新照片
- 使用 `Modifier.clip(RoundedCornerShape(16.dp))` 设置圆角
- 使用 `Modifier.size(80.dp)` 设置大小

### 6. 拍照按钮
- 外层3层圆环：使用 `Canvas.drawCircle()` 绘制
- 中层粉色圆形：使用 `Box` + `Modifier.background(Color(0xFFEC4899), CircleShape)`
- 内层库洛米头像：使用 `Text` 或 `Image`
- 呼吸动画：使用 `animateFloatAsState` 或 `rememberInfiniteTransition`
