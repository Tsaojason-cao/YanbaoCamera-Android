# 启动页设计分析

## 设计图: splash_screen.png

### 背景
- **渐变方向**: 垂直渐变（从上到下）
- **颜色**: 
  - 顶部: 深紫色 (#8B7FD8)
  - 中部: 紫粉混合 (#B89FE8)
  - 底部: 亮粉色 (#F5A8D4)
- **光晕球**: 多个白色半透明圆形，大小不一，随机分布

### 库洛米角色
- **位置**: 屏幕中央偏上
- **特征**: 
  - 黑色兔子耳朵（两个尖角）
  - 白色身体
  - 粉色鼻子和装饰
  - 黑色尾巴
  - 可爱表情
- **周围装饰**: 
  - 黄色星星（6个，分布在角色周围）
  - 白色光点（多个，散布在背景中）
  - 中央白色光晕（角色后方）

### 文字
- **标题**: "Yanbao Camera"
- **颜色**: 白色
- **字体**: 粗体，圆润风格
- **位置**: 屏幕中下部

### 进度条
- **位置**: 标题下方
- **样式**: 
  - 圆角矩形（圆角半径约 20dp）
  - 白色边框
  - 粉紫渐变填充（从左到右）
  - 半透明背景

### 动画效果（推测）
1. Logo缩放动画：0.8x → 1.0x（1秒）
2. 文字淡入：透明度 0 → 1（0.5秒）
3. 进度条加载动画（3秒）
4. 光晕球浮动动画（持续）
5. 星星闪烁动画（持续）

### 技术实现要点
1. 使用 Brush.verticalGradient 实现背景渐变
2. 使用 Canvas 绘制光晕球（多个 drawCircle，alpha 0.3-0.6）
3. 使用 Icon 或 Image 显示库洛米角色（需要矢量图或 PNG）
4. 使用 Text 显示标题，fontWeight = FontWeight.Bold
5. 使用 LinearProgressIndicator 或自定义 Canvas 绘制进度条
6. 使用 rememberInfiniteTransition 实现光晕球浮动动画
7. 使用 animateFloatAsState 实现 Logo 缩放动画
8. 使用 AnimatedVisibility 实现文字淡入动画
9. 使用 LaunchedEffect + delay(3000) 实现3秒后跳转
