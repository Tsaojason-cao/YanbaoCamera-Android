# 编辑主界面设计规格

## 设计图：EditAdjust_ToolsInterface_HighFidelity.png

### 顶部操作栏（Operation Bar）
- **左侧**：
  - ❌ 关闭按钮
  - 撤销按钮（Undo2）显示 "Xox 60%"
- **中间**：
  - 24px 文字大小指示器
- **右侧**：
  - 重做按钮（Redo2）显示 "Rox 20%"
  - ✅ 保存按钮（粉色，圆角矩形）

### 主画布区域（Main Canvas Area）
- **背景**：透明棋盘格
- **图片显示**：圆角矩形（渐变边框：粉色→紫色）
- **中间旋转图标**：手指点击旋转提示

### 底部编辑控制区（Editing Control Area）

#### 滑块控制部分（Slider Control Section）
- **标题**：BRIGHTNESS（当前选中工具）
- **滑块**：
  - 轨道：粉紫渐变（左）+ 白色（右）
  - 当前值：50%（右侧显示）
  - 轨道高度：6px
  - 滑块圆钮：16px
  - 轨道描述：`10px tracking-tighter white/40`

#### 工具滚动栏（Tool Scroll Bar）
8 个工具按钮（横向滚动）：
1. **Brightness（亮度）** - 选中状态
   - 图标：☀️ 太阳
   - 背景：粉紫渐变
   - 尺寸：80x80px
   - 圆角：24px
   
2. **Contrast（对比度）**
   - 图标：🌓 半月
   - 背景：灰色
   
3. **Saturation（饱和度）**
   - 图标：💧 水滴
   - 背景：灰色
   
4. **AI Enhance（AI增强）**
   - 图标：✨ 魔法棒
   - 背景：灰色
   
5. **Crop（裁剪）**
   - 图标：✂️ 剪刀
   - 背景：灰色
   
6. **Text（文字）**
   - 图标：Tp 文字
   - 背景：灰色
   
7. **Sticker（贴纸）**
   - 图标：😊 笑脸
   - 背景：灰色
   
8. **More（更多）**
   - 图标：...
   - 背景：灰色

### 技术规格
- **工具按钮间距**：24px（左）、16px（右）
- **工具按钮尺寸**：80x80px
- **工具栏高度**：180px
- **滑块区域高度**：562px
- **保存按钮**：200x40px，圆角 20px
- **底部装饰**：库洛米吉祥物（右下角）

### 交互逻辑
1. 点击工具按钮 → 切换当前编辑模式
2. 滑动滑块 → 实时更新参数值
3. 撤销/重做 → 显示操作历史百分比
4. 保存按钮 → 导出编辑后的图片
