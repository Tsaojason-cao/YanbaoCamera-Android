## 🚀 APK构建

### 构建方法（4种选择）

#### 方法1：Android Studio（推荐 - 最简单）
```
File → Open → Build → Build APK(s)
```
**优点**: 图形界面，易于使用
**时间**: 3-8分钟

#### 方法2：GitHub Actions（推荐 - 自动化）
```
git push origin main
# Actions自动构建
# 从Artifacts下载APK
```
**优点**: 自动化，无需本地SDK
**时间**: 5-15分钟

#### 方法3：Docker（完全隔离）
```
docker build -t yanbao-camera-builder .
docker run -v $(pwd)/output:/output yanbao-camera-builder
```
**优点**: 完全隔离，可重复
**时间**: 10-20分钟

#### 方法4：命令行（需要SDK）
```
./gradlew assembleRelease
```
**优点**: 快速，自动化
**时间**: 3-8分钟

### 推荐构建流程

**本地开发**:
```bash
gradle assembleDebug  # 快速迭代
```

**发布前**:
```bash
gradle assembleRelease  # 优化版本
jarsigner ...          # 签名
zipalign ...           # 对齐
```

**自动化**:
```bash
git push origin main   # 自动构建
# 从Actions下载APK
```
